# HumanOrNot

A Turing test in messenger format.

## Gameplay

- 5 minutes to chat with two entities (A and B)
- Choose from predefined responses
- Final verdict: which one is human?

## Features

- 10,000 unique dialogue nodes
- Two communication styles (formal / casual)
- Full localization (RU / EN)

## Important note

Dialogues are intentionally rough and not meant to be realistic.  
This project is a **technical demo** — it showcases architecture, data generation, localization, and UI, not natural conversation.

## Tech stack

HTML / CSS / JS / JSON

## Run locally

1. Generate `data.js`: `python build_data.py`
2. Open `index.html`
